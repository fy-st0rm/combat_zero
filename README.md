# Combat_Zero

This is a demo project to showcase 2D combat system using opengl in C

## Building
#### ⚠️ **Warning:** The build system is only supported for linux at this moment

1. **Bootstrap the builder:**
    ```sh
    g++ -o cbuild cbuild.cpp
    ```

2. **Run the cbuild:**
    ```sh
    ./cbuild
    ```

## Controls

| Key    | Action      |
|:------:|:-----------:|
|w       | Jump        |
|a       | Move left   |
|d       | Move right  |
|Space   | Dash        |
|Mouse 1 | Attack      |
|Esc     | Pause       |
